# C#移行の初期ベース

このディレクトリは、既存のC++実装をC#へ移行するための土台として追加したものです。

## 現在の移行状況

- プロトコル定数 (`src/ipmsg.h`) のC#移植
- 内部定数 (`src/ipmsgdef.h`) のC#移植
- 送受信ロジック（UDP/TCP）のベース実装
- プロトコル解析/生成の詳細仕様対応（IPDict形式、本文分解、型/ネスト）
- UI/ViewModelと設定管理のベース実装
- WPFによる送受信/設定/履歴/ログ/ログイン/プラグイン画面の具体的な画面実装
- ファイル送受信と暗号化処理のベース実装
- ファイル転送のメタデータ拡張/進捗管理/再送・中断/再開・復旧制御のベース実装
- 鍵交換/署名処理と鍵管理・互換性検証のベース実装
- .NET 10 向けにターゲットフレームワークを更新
